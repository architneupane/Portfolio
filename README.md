# Personal Portfolio Website

Welcome to my personal portfolio website! This project showcases my skills, projects. It is designed to be clean, user-friendly, and responsive across all devices.

## Features

- **Hero Section**: Animated text highlighting my roles and motivations.
- **Projects Section**: A showcase of my key projects with descriptions, technologies used, and links to GitHub repositories and live demos.
- **About Me**: A brief introduction to who I am, my skills, and my passion for development.
- **Resume**: A downloadable PDF of my resume.
- **Contact Form**: A functional form to send messages directly to me.
- **Social Links**: Quick access to my GitHub, LinkedIn, and Facebook profiles.

## Technologies Used

- **HTML5**: For the structure of the website.
- **CSS3**: For styling and layout, including animations and responsive design.
- **JavaScript**: For interactivity, such as the animated text in the hero section.
- **Font Awesome**: For social media icons.

## How to Use

1. Clone the repository:
   ```sh
   git clone https://github.com/architneupane/portfolio.git
   ```
2. Navigate to the project directory:
   ```sh
   cd portfolio
   ```
3. Open `index.html` in your browser to view the website locally.
